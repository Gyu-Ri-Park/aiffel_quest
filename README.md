[O]  **2. 전체 코드에서 가장 핵심적이거나 가장 복잡하고 이해하기 어려운 부분에 작성된

[O]  **3. 에러가 난 부분을 디버깅하여 문제를 “해결한 기록"을 남겼거나 "새로운 시도

- 각 루브릭에 대해 입금된 금액 정보나 현재 잔액, 출금 정보, 현재 잔고 등의 정보를 함께 제공해주는 점이 좋았습니다.
